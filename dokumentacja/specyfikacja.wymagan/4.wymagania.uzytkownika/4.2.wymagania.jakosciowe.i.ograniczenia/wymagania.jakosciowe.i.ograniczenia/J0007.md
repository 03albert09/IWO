#### (J0007) Maksymalny czas awarii (MTTR)
|                   |                                                                                                                         |
| ----------------: | :---------------------------------------------------------------------------------------------------------------------- |
| **Typ** | *Niezawodność* |
| **Priorytet** | *krytyczne* |
| **Trudność** | *wysoka* |
| **Sposób pomiaru** | Pomiar czasu usunięcia awarii losowych lub symulowanych awarii |
| **Oczekiwane wartość** | Średni maksymalny czas naprawy awarii (MTTR) wynosi 2 godziny |
